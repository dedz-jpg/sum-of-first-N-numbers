Descrição do problema

Calcular a soma dos números pares entre 1 e N.
Exemplo

```python
N = 12
soma = 0
i = 1

while i <= N:
    if i % 2 == 0:
        soma = soma + i
    i = i + 1

print(soma)


Conceitos usados

    Variáveis

    Loop while

    Condicional if

    Operador módulo %

    Incremento de variáveis
